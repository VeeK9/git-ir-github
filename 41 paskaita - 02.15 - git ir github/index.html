Git

Git'as yra populiariausia kodo versijavimo programa.

Git komandos

  • init - sukuria git'o repozitorija dabartiniame aplankale
  • status - nurodo informacija apie lokalios git'o repozitorijos statusa
  • add - prideda nurodytus failus prie commitui paruostu (stage'ina)
  • rm --cached - isima nurodytus failus is stage'inimo zonos
  • commit -m "zinute"
